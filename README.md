# Nia App

## Project Documentation

### Overview
Nia App is an Android application designed to provide an intuitive interface for users to manage their tasks effectively.

### Features
- User-friendly interface
- Task categorization
- Reminder notifications

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/jimenezcristal769-byte/android-app.git
   ```
2. Open the project in Android Studio.
3. Build the project and run the app on an Android device or emulator.

### Usage
- Open the app, and you'll be greeted with a dashboard displaying your tasks.
- Add, edit, and delete tasks as needed.
- Set reminders to stay on track with your tasks.
